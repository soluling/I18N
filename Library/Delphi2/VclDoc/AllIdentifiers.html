See Samples\Delphi\VCL\DualLanguage sample to see how to use the event.

NtBeforeTranslateEx NtBaseTranslator An event that is called before translating a property value. Use this to disable or change the translation process. If you assign this value, make sure that the event is as fast as possible because this event is called on every single property of the application.

See Samples\Delphi\VCL\DualLanguage sample to see how to use the event.

initialization
  NtEnabledProperties := STRING_TYPES;
end.
